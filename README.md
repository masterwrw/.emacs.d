# 自定义的常用快捷键
| key | describle         |
|-----+-------------------|
| C-f | swiper            |
| C-s | save              |
| M-o | open file         |
| C-o | dired             |
| C-q | expand region     |
| M-y | popup kill ring   |
| M-g | go to char        |
| M-l | go to line        |
| cc  | copy word         |
| cl  | copy line         |
| dd  | delete line       |
| M-n | new next line     |
| M-p | new previous line |
